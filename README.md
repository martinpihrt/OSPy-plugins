# OSPy-plugins
Python - OpenSprinkler Py (OSPy) plugins

Theses I2C addresses use in available plugins:
	air_temp_humi 	0x03
	tank_monitor 	0x04
	humi_monitor 	0x05
	button_control	0x27
	lcd_display 	0x20-0x27, 0x38-0x3F
	real_time 		0x68
	volt_temp_da 	0x48
	water_meter 	0x50, 0x51
	wind_monitor 	0x50, 0x51
